# Python Internship Projects

## Description
This repository contains Python projects completed as part of my Python Programming Internship. These projects demonstrate basic Python programming concepts such as user input, loops, lists, and calculations.

## Projects

### Task 1: To-Do List
A simple application that allows users to:
- Add multiple tasks
- Store tasks in a list
- Display the complete to-do list with numbering

### Task 2: Expense Tracker
A simple application that allows users to:
- Enter multiple expenses
- Store expenses in a list
- Calculate and display the total amount spent

## Technologies Used
- Python 3

## Files
- `task1.py` – To-Do List Application
- `task2.py` – Expense Tracker Application

## How to Run
1. Clone or download this repository.
2. Open the project folder in a terminal.
3. Run the desired file:
   ```bash
   python task1.py
   ```
   or
   ```bash
   python task2.py
   ```
